# Learning

Starter の内部実装を理解するための学習教材です。

---

## Learning の目的

この教材は、Starter Package（`report()` や `slide()`）が内部で行っている設定を、
**段階的に** 理解するために作られています。

Learning では **Package を使いません**。
純粋な Typst のコードのみで書かれており、各ファイルが独立してコンパイルできます。

---

## 推奨学習順

01 → 08 の順に読むことを推奨します。
各ファイルは**前段階との差分のみ追加**する方式で書かれています。

| ファイル | 学習内容 | 追加される知識 |
|---|---|---|
| `01_minimal.typ` | 最小のドキュメント | Typst ファイルの基本 |
| `02_page.typ` | ページ設定 | `set page()`, `set par()`, 余白, ページ番号 |
| `03_text.typ` | フォント設定 | `set text()`, フォントフォールバック, 言語設定 |
| `04_heading.typ` | 見出し設定 | `set heading()`, `show heading`, ナンバリング |
| `05_figure.typ` | 図表・数式設定 | `set figure()`, supplement, 数式番号 |
| `06_table.typ` | 表の詳細 | 列幅指定, ストライプ, 罫線 |
| `07_report_template.typ` | テンプレート化 | `#let` 関数, `show:` ルール, テンプレートの構造 |
| `08_slide_template.typ` | スライド基礎 | 16:9 ページ, ゴシック体, ページ区切り |

---

## Package との違い

| | Learning | Package |
|---|---|---|
| 目的 | **学習・理解** | **利用** |
| Package 利用 | ❌ 使わない | ✅ これ自体が Package |
| コメント量 | 多い（教育重視） | 少ない（保守性重視） |
| 構成 | 1 ファイルに全設定 | モジュール分割 |
| 対象 | 学習者 | 利用者 |

---

## コンパイル方法

各ファイルは独立してコンパイルできます。

```powershell
typst compile learning/01_minimal.typ
```

VSCode で開くと自動的にプレビューされます。

---

## 学習のゴール

08 まで読み終えると、以下が理解できます：

- Typst の `set` ルールと `show` ルールの違い
- フォントのフォールバック設定の仕組み
- 見出し・図表・数式のカスタマイズ方法
- テンプレート関数の作り方

この知識があれば、Starter Package のソースコードも読めるようになり、
将来的に自分自身でテンプレートをカスタマイズすることも可能です。
