# Typst Starter 要求仕様書
Version: 1.0.0 (Draft)

---

# 1. 概要

## 1.1 プロジェクト名

Typst Starter（仮称）

## 1.2 プロジェクトの目的

Typst Starter は、Windows 環境において Typst を初めて利用する大学生や研究室の学生を対象に、Typst の導入から最初のレポート作成までを支援するスターターキットである。

Typst 本体の開発やリファレンスの提供を目的とするものではなく、

> 「誰でも短時間で Typst によるレポート作成を開始できる」

ことを目的とする。

---

# 2. 開発理念

本プロジェクトは以下の理念を持つ。

## 2.1 公式ドキュメントを尊重する

Typst 公式ドキュメントは非常に充実している。

本プロジェクトは公式ドキュメントを置き換えるものではなく、公式ドキュメントへ到達するまでの橋渡しを行う。

そのため、Typst の文法や関数をすべて説明することは目的としない。

---

## 2.2 学習を妨げない自動化

初心者が繰り返し苦労する部分は支援する。

一方で、

- VSCode の基本操作
- 拡張機能の導入
- GitHub の利用方法

など、今後の学習に役立つ部分については、自動化ではなく分かりやすいドキュメントを提供する。

---

## 2.3 初心者を対象とする

本プロジェクトは

- VSCode をほとんど使ったことがない
- GitHub を使ったことがない
- Typst を初めて使う

利用者を対象とする。

---

## 2.4 Windows を対象とする

Version 1.0.0 では Windows を正式対応とする。

Linux および macOS は将来対応予定とする。

---

## 2.5 自立した利用者を育てる

本プロジェクトは、環境構築を容易にするだけではなく、
利用者が将来的に自分自身で Typst を学び、
テンプレートを理解し、
公式ドキュメントを活用できるようになることを目指す。

そのため、必要以上のブラックボックス化は行わず、
学習機会を残した設計を採用する。

---

# 3. 対象利用者

対象利用者は以下とする。

- 大学生
- 大学院生
- 研究室配属学生
- Typst 初学者
- VSCode 初学者

主に研究室内での利用を想定する。

---

# 4. プロジェクトの責任範囲

Typst Starter が責任を持つ範囲は、

「Typst を利用したレポートを書き始められる環境」

までとする。

具体的には

- Typst の導入方法
- VSCode の導入方法
- 必要な拡張機能
- 推奨設定
- 日本語フォントの紹介
- テンプレートパッケージ
- サンプルプロジェクト

を提供する。

---

# 5. 責任範囲外

以下は本プロジェクトの責任範囲外とする。

## Typst のリファレンス

Typst の文法

関数

レイアウト

数式

図表

これらは公式ドキュメントを参照する。

---

## Git の教育

Git の詳細な使い方

ブランチ

Pull Request

などは扱わない。

GitHub の導入のみ紹介する。

---

## Typst の応用

独自テンプレートの設計

パッケージ開発

高度なレイアウト

などは扱わない。

# 8. 利用フロー

利用者は以下の流れで利用する。

README を読む

↓

Typst をインストールする

↓

VSCode を導入する

↓

推奨拡張機能を導入する

↓

必要に応じて Harano Aji フォントを導入する

↓

Typst Starter Package を導入する

↓

Project Template を利用して最初のレポートを書く

↓

公式ドキュメントを利用しながら学習する

---

# 9. Typst_Template との関係

Typst Starter は初心者向けプロジェクトである。

一方、

Typst_Template はテンプレート開発用リポジトリとして維持する。

Typst Starter は Typst_Template の安定版を利用し、教育・配布を目的とした構成とする。

# 11. 本プロジェクトのゴール

本プロジェクトの最終目標は、

「大学生が Typst の環境構築で躓くことなく、
短時間で最初のレポートを書き始められること」

である。

Typst を教えることではなく、

Typst を学び始められる環境を提供することが本プロジェクトの使命である。
